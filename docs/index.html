0.1 Important notes

This is a free textbook teaching introductory statistics for undergraduates in psychology. This textbook was adapted from a larger OER course package for teaching undergraduate statistics in psychology, of which all materials are free and copiable, with source code maintained in Github repositories. The main OSF project link is https://osf.io/3s68c/.

All resources are released under a creative commons licence CC BY-SA 4.0. Click the link to read more about the license, or read more below in the license section.

Team members contributing new content include Matthew Crump, Alla Chavarga, Anjali Krishnan, Jeffrey Suzuki, and Stephen Volz. Much of the text was primarily written by Matthew J. C. Crump. Jeff contributed the YouTube videos peppered throughout the textbook. All of Jeff’s statistics videos are available on his Youtube channel: Statistics Video playlist.

0.2.2 Hypothes.is

Hypothesis is a web-browser plug-in that lets you make comments on websites by highlighting text, and then making comments. Feel free to use hypothesis with this textbook. We will read your comments.

  1. Use Hypothes.is, an amazing tool for annotating the web.

  1. Go to Hypothes.is, and “get-started”

  2. Install the the add-on for chrome, or other browser

  3. That’s it, turn on Hypothes.is when you are reading this textbook, and you will see all public annotations made by anyone else.
